# 配色工坊 | Palette Studio

互動式配色主題展示工具，提供多種精選色彩主題，搭配即時 UI 預覽，幫助設計師與開發者快速挑選合適的配色方案。

## 功能特色

- 多種配色主題分類（經典、自然、現代、柔和）
- 即時 UI 預覽（含側邊欄、卡片、表格、按鈕等元件）
- 色階展示與 CSS 變數輸出
- 支援亮色／暗色模式切換
- 莫蘭迪色系專屬頁面

## 使用方式

直接在瀏覽器中開啟 `index.html` 即可使用，無需安裝任何相依套件。

## 檔案結構

```
index.html      # 主頁面，包含完整配色主題瀏覽與預覽功能
morandi.html    # 莫蘭迪色系專屬展示頁面
```
